Exercises

Exercise 1

The operator fromEvent should help you to log "Hello world!" each time you click on the button (only).

Exercise 2

The operator map should help you to log the date each time you click on the button (only) with Date.now().

Exercise 3

The operator delay should help you to log the date each time you click on the button (only) with a delay of 2 seconds.

Exercise 4

The operator merge should help you to log the date each time you click on the button or the checkbox, but always with a delay of 2 seconds with the button (only).

Exercise 5

The operator interval should help you to create a new stream to log the date every 2 seconds.
